# Panoptimon

The All-Seeing System Monitor Daemon

## Installation

Add this line to your application's Gemfile:

    gem 'panoptimon'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install panoptimon

## Usage

  panoptimon --help
